Project Lead • Technical Program Manager

Led initiatives and teams for the French national electricity project Linky

Project Details

  • Client Type : Large enterprise
  • Industry : Electricity
  • Location : France
  • Started on : April 2014
  • Duration : 1 year 3 months

Description

Led initiatives for the French national electricity project Linky (~100 people). This countrywide project aimed to replace all the standard electricity meters with remotely operated ones.

Achievements

  • • Ensured continuous value delivery by fostering collaboration within the engineering unit (including the engineering managers, architects, and developers) and between relevant functions (e.g. leadership, product, engineering).

  • • Collaborated with the Agile coach to roll out Agile methods across teams.

  • • Provided guidance on software best practices.

  • • Analysed and implemented new features in various areas of the product.

  • • Enabled the rollout of the first several million smart meters in different French regions.
